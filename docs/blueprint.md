# **App Name**: GridSystem Portfolio

## Core Features:

- Rigid Header Navigation: High-contrast global header with a 2px black border, featuring a square-boxed logo, central nav links, and a prominent 'Hire Me' button.
- Geometric Footer: Matching high-contrast footer with zero-radius corners containing site-map and social connectivity links.
- AI Dev-Pitch Generator: An AI-powered tool that analyzes your project data and generates a compelling 50-word introduction based on the portfolio content.
- Dynamic Project Showcase: A filterable grid for web projects following the stark 2px black border and square-corner visual language.
- Open-Source Package Registry: A custom layout specifically for showcasing npm or other packages with installation command-line blocks.
- Lead Capture System: High-visibility contact form with geometric input fields for client inquiries.

## Style Guidelines:

- Primary Color: Deep Cobalt (#1B32A3) used for CTA highlights to contrast with the high-intensity black and white design.
- Background: Strict off-white (#FAFAFA) to maintain a sterile, modern, and architectural feeling.
- Accent Color: Stark Violet (#30109C) used for interactive states like hover effects on links and rigid boxes.
- Headline font: 'Space Grotesk' for its techy and scientific feel, reinforcing the geometric structure. Body font: 'Inter' for a clean, machine-neutral reading experience.
- Minimalist, thin-line icons (2px stroke width) with no rounded corners to match the border styles.
- Grid-based architecture with absolute 'rounded-none' constraints and 2px solid black (#000000) borders dividing all functional blocks.
- Instant or stepped transitions (0ms or snap-motion) to preserve the rigid, mechanical feeling of the layout.
